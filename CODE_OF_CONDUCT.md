# Code of Conduct

We want this project to be useful to newcomers, researchers, educators, and contributors from many backgrounds.

Be respectful, specific, and evidence-focused. Critique ideas and methods rather than people. Welcome basic questions. Avoid harassment, personal attacks, discrimination, and deliberately misleading claims.

Project maintainers may edit or remove contributions that make the community unsafe or materially reduce the reliability of the project.

To report a conduct concern privately, contact the repository owner through their GitHub profile.
