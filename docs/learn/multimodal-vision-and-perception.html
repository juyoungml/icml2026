Multimodal, Vision, and Perception

A multimodal model should use what it sees, not merely produce a plausible sentence from language patterns.

Start with intuition

The mental model

The system converts images, video, audio, or 3D observations into representations, aligns them with language, and then reasons or acts over the combined evidence.

Inside this area

Vision-language reasoning; cross-modal alignment; video and motion; 3D and spatial understanding; robustness and adversarial perception.

Common methods

Vision-language post-training; multimodal transformers; temporal modeling; 3D geometry; video diffusion; grounding and attribution.

Technical core

A common research pipeline

  1. 1Encode visual or temporal observations
  2. 2Align observations with language or actions
  3. 3Fuse information across space and time
  4. 4Evaluate grounding, consistency, and robustness—not only answer accuracy
Grounded predictionprediction = f(text tokens, visual tokens, time, geometry)

A convincing test changes or removes relevant visual evidence and checks whether the prediction changes appropriately.

Make it concrete

Worked example: motion versus appearance

  1. Two training clips show the same object but different motion quality.
  2. An attribution method weights temporal regions rather than static pixels.
  3. The method ranks which clips influence generated motion.
  4. Human preference is helpful, but physical plausibility needs separate evaluation.
Pause and predict

Can the method localize the visual evidence behind an answer?

Show a strong answer

Look for evidence tied to image and video reasoning; spatial consistency; temporal dynamics; localization; robustness under corruptions and distribution shift. Then check whether the result survives this limitation: Models can answer through language priors without using visual evidence, while aggregate vision labels hide large differences between video, 3D, and robustness work.

Do not overclaim

Models can answer through language priors without using visual evidence, while aggregate vision labels hide large differences between video, 3D, and robustness work.

Evaluation

How progress is tested

Image and video reasoning; spatial consistency; temporal dynamics; localization; robustness under corruptions and distribution shift.

Useful comparison

Image classification asks what is present. Grounded reasoning asks which evidence supports an answer. Video and 3D additionally require temporal or spatial consistency.

Paper 1 · Core caseOutstanding Paper Honorable Mention · oral

Motion Attribution for Video Generation

Problem
Video generators improve quickly, but it is unclear which fine-tuning clips cause better or worse motion rather than merely changing appearance.
Method
Motive uses gradient-based data attribution with motion-weighted loss masks to isolate temporal influence from static appearance.
Evidence
Text-to-video fine-tuning, VBench motion measures, data-curation experiments, and a reported 74.1% human-preference win rate over the base model.
Main result
High-influence clips selected by motion attribution improve temporal consistency, physical plausibility, smoothness, and dynamic degree.
Important limit
Attribution depends on the chosen motion loss and model; human preference does not fully establish causal or physical correctness.

Paper 2 · Abstract-based previewComparison case

Holi-Spatial: Evolving Video Streams into Holistic 3D Spatial Intelligence

Why selected: Spatial-data scaling broadens the core case and shows another important part of this area.

Problem
Existing spatial datasets are small, manually curated, and poorly matched to the diversity of raw web video.
Approach
Automatically turn video into reconstructed 3D scenes, geometric annotations, grounding examples, and spatial question-answer pairs.
Evidence
A 4-million-example resource, reconstruction-quality comparisons, and VLM fine-tuning on spatial reasoning tasks.
Takeaway
The pipeline aims to make detailed 3D supervision scalable while improving downstream spatial reasoning.
Caution
Automatically produced labels can inherit reconstruction errors; dataset scale does not by itself establish physical understanding.
Metadata cues
LLM post-training; Reasoning / test-time compute; Agents / tool use; Graphs / geometry · vision/video; language/llm

Paper 3 · Abstract-based previewComparison case

Bad Seeing or Bad Thinking? Rewarding Perception for Multimodal Reasoning

Why selected: Perception/reasoning credit assignment broadens the core case and shows another important part of this area.

Problem
When a VLM fails, ordinary rewards do not reveal whether perception or later reasoning caused the error.
Approach
Split perception from reasoning and route structured rewards using a blindfolded-reasoning proxy and modality-aware credit assignment.
Evidence
Free-form vision-language tasks designed to measure whether targeted perception rewards improve both stages.
Takeaway
The work argues that more precise credit assignment can reduce the trade-off between seeing accurately and reasoning well.
Caution
Proxy verification may still reward incomplete visual evidence, so intervention-based grounding tests remain important.
Metadata cues
RL / policy optimization; Reasoning / test-time compute; Agents / tool use · vision/video; language/llm
